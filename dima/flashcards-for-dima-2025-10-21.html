How many data types does JavaScript have?
8 in total. 7 primitives and the Object data type.
Is the following line of code correct?
let message;
Yes. But the value will be undefined.
Is the following line of code correct?
let "message";
No. You cannot start the name of a variable with double quotes.
Is the following line of code correct?
let "message" = hello;
No. You cannot start the name of a variable with double quotes. The word hello should be the one in quotes.
Is the following line of code correct?
let message = "hello";
Yes
