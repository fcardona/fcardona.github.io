How to host a website totally free

Most businesses pay a monthly or yearly fee to have their website on the internet. However, a great number of those websites can be hosted for free. As a matter of fact, right now I pay $0.00 for hosting felixcardona.com. In the next few days I will show you how to create a basic website and host it online totally free of charge. Lets begin with a brief discussion about metaphors.

Computers are full of metaphors

The main screen that you see on a Windows computer full of icons and a background color or image is called the "desktop". When you delete a file it ends up in the "trash bin". It uses terms related to an office environment as a metaphor.

The field of computer networks and the internet is also full of metaphors. In this case that of a restaurant. You have the host, servers, and clients. The host is a computer or operating system that has one or more servers at its disposal. One type of server is a web server. Just like your food server at a restaurant is ready to fulfill your food requests, a web server is software installed on a host that is ready to fulfill a clients request for a web page. And that brings us to clients. A client is a software that makes a request from a server. Google Chrome, Microsoft Edge, Firefox, and Safari are all web clients. Their primary role is to request web pages from web servers.

HTML, the markup language of the web

One subset of computer languages are called markup languages. Their name stems from the fact that they are used to mark or identify different parts of a text document. For example, a typical document may have a title, a headline, the body, an image, etc. HTML is the markup language of the web. And you are about to create your first webpage.
